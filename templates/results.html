Gender Classifier

Using Machine Learning to Classify Gender

How does it achieve the same? Ling Application uses Machine Learning and Statistical Tools to build a Model which can predict your Gender. A Flask Micro-Framework is used to create an Application Programming Interface (API) which accepts local HTTP Requests and deploy our Application.
